# Style Cloth

Sistema de escritorio desarrollado en **Java (Swing)** para la gestión básica de productos en una boutique de moda.  
Proyecto académico realizado en equipo, enfocado en programación orientada a objetos, manejo de base de datos y desarrollo de interfaces gráficas.

## Descripción
Style Cloth es un sistema de escritorio diseñado para apoyar la administración de una boutique de moda, permitiendo el control de productos, ventas, compras, clientes, empleados y proveedores.

## Funcionalidades
- Inicio de sesión básico.
- Registro, consulta, actualización y eliminación de productos (**CRUD**).
- Interfaz gráfica desarrollada con **Java Swing**.
- Validación de datos de entrada.
- Organización del código mediante clases y paquetes.
- Flujo completo de interacción usuario–sistema.

## Tecnologías utilizadas
- **Lenguaje:** Java  
- **Interfaz gráfica:** Java Swing  
- **IDE:** Eclipse  
- **Control de versiones:** Git & GitHub  

## Base de datos
El sistema utiliza una base de datos relacional en **SQL Server** para el almacenamiento de la información.  
La conexión se realiza mediante JDBC y se incluyen configuraciones para la gestión de datos.

## Ejecución del proyecto
1. Abrir **Eclipse IDE**.
2. Importar el proyecto:
   - `File` → `Import` → `Existing Projects into Workspace`.
3. Seleccionar la carpeta del proyecto.
4. Ejecutar la clase principal del sistema.
