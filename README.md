# Style Cloth

Sistema de gestión para una boutique de moda, desarrollado como proyecto académico en equipo.

## Descripción
Style Cloth es un sistema de escritorio diseñado para apoyar la administración de una boutique de moda, permitiendo el control de productos, ventas, compras, clientes, empleados y proveedores.

## Funcionalidades
- Inicio de sesión y gestión de usuarios
- CRUD de productos
- Gestión de ventas y compras
- Administración de clientes y proveedores
- Gestión de empleados
- Generación de reportes
- Respaldo de información

## Tecnologías utilizadas
- Java
- SQL Server
- Eclipse IDE

## Base de datos
- Base de datos relacional en SQL Server
- Scripts SQL incluidos para creación de tablas y relaciones
